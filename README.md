## 🧩 About the Application

-Project-Nord-Spiking-Neural-Network-Language-Model is a new type of language program. It uses Spiking Neural Networks (SNN), a different way to mimic brain activity. The model has 144 million parts and operates with 97% sparsity. It can run on your phone and learns while you use it, making its responses more personal over time. The model was created with a small training budget, showing efficiency in design and use.

This app is for anyone interested in testing a unique language model that learns differently and uses less energy. You do not need advanced technical skills to use it.
